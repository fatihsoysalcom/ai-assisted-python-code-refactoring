# AI Assisted Python Code Refactoring

This example demonstrates how AI tools can assist developers by suggesting more concise and Pythonic code solutions. It shows a common coding problem solved first with a traditional, multi-step approach, and then with an AI-suggested, more efficient list comprehension, highlighting AI's role as a learning catalyst.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
